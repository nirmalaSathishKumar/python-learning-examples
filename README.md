# Python Learning Examples

A collection of executable Python examples covering fundamental concepts from "Python for Everybody" course.

## Structure

Each folder contains small, executable Python scripts demonstrating specific concepts:

### 01-variables
- `basic_variables.py` - Variable declaration and basic types
- `arithmetic_operations.py` - Mathematical operations

### 02-conditionals
- `if_else.py` - Conditional statements
- `logical_operators.py` - Boolean logic (and, or, not)

### 03-functions
- `basic_functions.py` - Function definition and calling
- `default_parameters.py` - Functions with default arguments

### 04-loops
- `for_loop.py` - For loop iterations
- `while_loop.py` - While loop examples
- `break_continue.py` - Loop control statements

### 05-strings
- `string_basics.py` - String indexing and slicing
- `string_methods.py` - Common string methods

### 06-lists
- `list_basics.py` - List creation and manipulation
- `list_operations.py` - Sorting, reversing, min/max
- `list_comprehension.py` - List comprehensions

### 07-dictionaries
- `dict_basics.py` - Dictionary operations
- `dict_iteration.py` - Looping through dictionaries

### 08-tuples
- `tuple_basics.py` - Tuple creation and unpacking

### 09-files
- `file_write_read.py` - File I/O operations
- `file_readline.py` - Reading files line by line
- `file_error_handling.py` - Exception handling with files

### 10-exceptions
- `basic_exception.py` - Try-except blocks
- `try_else_finally.py` - Complete exception handling

### 11-regex
- `regex_basics.py` - Regular expression patterns
- `email_validation.py` - Email validation with regex

### 12-classes
- `basic_class.py` - Class definition and objects
- `inheritance.py` - Class inheritance

### 13-web-scraping
- `url_request.py` - Basic URL requests with urllib

### 14-json
- `json_basics.py` - JSON encoding/decoding
- `json_file.py` - Reading/writing JSON files

### 15-databases
- `sqlite_basics.py` - SQLite database operations

## Usage

Each file can be run independently:

```bash
python 01-variables/basic_variables.py
python 04-loops/for_loop.py
python 12-classes/basic_class.py
```

## Requirements

All examples use Python standard library only - no external dependencies required!

## Notes

- All code is executable and production-ready
- No comment lines - code is self-explanatory
- Files are small and focused on single concepts
